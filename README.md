# NZX0 - ᴘᴛ

NZX0 é um projeto pessoal de estudos criado para experimentar tecnologias modernas de desenvolvimento web e aperfeiçoar minhas habilidades.
O objetivo não é desenvolver um produto final, mas explorar conceitos, animações, experiências em 3D, técnicas de renderização e boas práticas através da prática.
Este repositório funciona como um laboratório onde posso testar ideias, criar protótipos e evoluir continuamente como desenvolvedor.

## Objetivos

- Aprender animações avançadas com GSAP
- Explorar Three.js e renderização 3D
- Melhorar arquitetura em React
- Praticar desenvolvimento frontend moderno
- Experimentar interfaces criativas
- Testar técnicas de otimização de performance
