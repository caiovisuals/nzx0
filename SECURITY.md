# SEGURANÇA - ᴘᴛ

Embora este seja um projeto pessoal de estudos, deve sempre levar a sério qualquer questão de segurança. Este é um projeto de aprendizado em desenvolvimento contínuo. Apenas a versão mais recente da branch principal recebe correções de segurança.

## Escopo

Por se tratar de um projeto sem ambiente de produção oficial, o foco está em:

- Dependências vulneráveis (Next.js, React, Three.js, GSAP e outras)
- Exposição acidental de segredos ou credenciais no código
- Configurações inseguras de build ou de runtime

Itens fora do escopo incluem ataques de negação de serviço (DoS), engenharia social e relatórios automatizados sem prova de conceito.

# SECURITY - ᴇɴ

Although this is a personal study project, security concerns should always be taken seriously. This is a learning project under continuous development. Only the latest version of the main branch receives security fixes.

## Scope

As this project lacks an official production environment, the focus is on:

- Vulnerable dependencies (Next.js, React, Three.js, GSAP, and others)
- Accidental exposure of secrets or credentials in the code
- Insecure build or runtime configurations

Out-of-scope items include denial-of-service (DoS) attacks, social engineering, and automated reports lacking a proof of concept.
